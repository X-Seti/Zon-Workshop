# Zon Workshop

GTA SA zone and occluder file editor with radar map overlay.

Base: [RadarWorkshop](https://github.com/X-Seti/Img-Factory-1.6) — radar tile viewer.
Part of the IMG Factory 1.6 modding suite.

## Planned Features
- Radar map display (SA 1,296 tiles)
- `.zon` file parsing and overlay
- Occluder zone display and editing
- Zone type colour coding

## Requirements
```
pip install PyQt6
```

## Usage
```bash
./zon_workshop.py
```
